# 波形君 開発のポイントと設計思想

「波形君」は、エンジニアが直感的にタイミングチャートを作成・共有できるツールとして開発されました。以下にその主要なポイントをまとめます。

## 1. コア機能：WaveDrom互換の直感的な記法
- **テキストベースの波形定義**: `~` (High), `_` (Low), `p` (Clock) などの記号を組み合わせるだけで、複雑な波形を即座に描画できます。
- **リアルタイムレンダリング**: 設定を変更すると即座にSVG波形が更新されるため、試行錯誤が容易です。
- **拡張構文**: `r` (High+点線), `f` (Low+点線) などの独自記法により、信号の立ち上がり・立ち下がりを強調できます。

## 2. データの永続性とポータビリティ
- **IndexedDB による自動保存**: 編集中のデータはブラウザ内に自動保存されます。不意にブラウザを閉じても、作業内容は失われません。
- **ストレージ永続化**: ブラウザによるデータの自動削除を防ぐ `navigator.storage.persist` を実装し、信頼性を高めています。
- **JSON インポート/エクスポート**: 作成した波形データはJSON形式で保存・共有が可能です。

## 3. 高度な描画オプション
- **データバス表示**: `0-9` の数値でバスの色分けが可能。
- **スロープ（傾斜）描画**: デジタル信号特有の立ち上がり時間を表現可能。
- **表示倍率調整**: チャート全体の横幅を 100%〜200% の範囲で調整でき、長大な波形も詳細に確認できます。

## 4. 開発・デプロイ戦略
- **Local-First PWA**: オフラインでも動作し、スマホへのインストールも可能です。
- **GitHub Pages 自動デプロイ**: GitHub Actions を利用し、リポジトリへのプッシュのみで最新版が公開される仕組みを構築しています。
- **モダンなスタック**: React + Tailwind CSS + D3.js を採用し、高速でメンテナンス性の高いコードを実現しています。
